=====
ProTK
=====

:Version: 0.3.0-DEVELOPMENT
:Updated: 15 Feb 2012

Overview
========
ProTK is a speech processing toolkit integrating HTK, Praat, and WEKA to form
models for machine learning and analysis of large sets of speech data.
